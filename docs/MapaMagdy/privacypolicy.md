## Privacy Policy
### Data collection
Mapa Magdy application does not collect any personal data. Application requests from user only his location data that is being used only to locate current user position and calculate route to seleted point of interest. Location data is NOT collected, and NOT connected to your identity, and NEVER used for advertisement, or other activities except getting current user position. Application gathers application usage events statistics and sends those to Google's Firebase Analytics under author's account. This information NOT connected to user identity, and used by application author in order to improve user experience and further development of application, this information NOT provided and NOT used for advertisement purposes. 

### Data storage
Application data is stored locally on your device. Images are downloaded from author's Firebase storage.
